Aula 02 -- Programação Web

Comandos necessários para ativar todas propriedades do express:
Instala node_modulos e jsons   ==>  npm install node-modules
Instala cria o json  ==>  npm init -y
Instala o express    ==>  npm i express
Ativar o servidor    ==>  npm i --save-dev nodemon 

npx nodemon app ==> rodar servidor

(localização)localhost:3000 --> simulando no navegador, roda as informações apenas nesse momento, devemos sempre recarregar

Extensão sqlite3	   ==> npm i better-sqlite3
Instalar KNEX        ==> npm knex init
Instalar CORS        ==> npm install cors
